## 🎨 设计资源

### Logo设计要求
```
- 简洁现代的设计风格
- 体现AI和面试的概念
- 适用于各种尺寸（16x16到512x512）
- 包含深色和浅色版本
- 提供SVG、PNG格式
```

### 配色方案
```
主色：#4F46E5 (蓝紫色)
辅色：#059669 (绿色)
警告：#DC2626 (红色)  
中性：#6B7280 (灰色)
背景：#F9FAFB (浅灰)
```

### 截图规范
```
- 分辨率：1920x1080或更高
- 包含浏览器框架
- 突出核心功能界面
- 使用真实但脱敏的数据
- 保持界面整洁美观
```
