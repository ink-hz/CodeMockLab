## 🎯 截图标准

### 质量要求
- **分辨率**: 至少1200x800px
- **格式**: PNG (截图) / GIF (动画)
- **文件大小**: PNG < 500KB, GIF < 10MB
- **清晰度**: 高清无模糊

### 内容要求
- **数据**: 使用脱敏的演示数据
- **界面**: 保持整洁美观
- **功能**: 突出核心功能点
- **一致性**: 保持设计风格统一
